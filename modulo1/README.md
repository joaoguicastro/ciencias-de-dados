# Módulo 1 - Fundamentos de Python

Esse é o modulo introdutório do Curso de Python para Ciência de Dados. Aqui você aprenderá sobre as partes mais básicas da linguagem Python:

- Sintaxe
- Variáveis
- Estruturas de Dados
- Estruturas Condicionais
- Laços de Repetição

## Instruções para executar o código

### No Google Colab

Você pode abrir os notebooks diretamente no Google Colab de duas formas:

- Abrir do GitHub:
	- Acesse https://colab.research.google.com/github/gustavonogvi/curso-python-ciencia-de-dados/blob/main/modulo1_fundamentos_python.ipynb

- Fazer upload local:
	1. Acesse https://colab.research.google.com
	2. Clique em "Upload" e selecione o arquivo `modulo1_fundamentos_python.ipynb` ou `exercicios_modulo1.ipynb`.

- Crie uma cópia do notebook se não conseguir executá-lo.

### Na sua máquina

Primeiro, clone o repositório:

```bash
git clone https://github.com/gustavonogvi/curso-python-ciencia-de-dados.git
cd curso-python-ciencia-de-dados
```

Crie um ambiente virtual e ative-o:

```bash
python3 -m venv .venv
source .venv/bin/activate  # para Linux e MacOS
.\.venv\Scripts\activate  # para Windows
```

Instale as dependências:

```bash
pip install jupyter
```

Inicie um servidor Jupyter:

```bash
jupyter notebook
```

Acesse o servidor em:

`http://localhost:8888`
