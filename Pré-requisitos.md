# Pré-requisitos

## Pré-requisitos Essenciais**
- **Linha de comando básica**: navegar em pastas, executar scripts, instalar pacotes.
- **Matemática e estatística**: álgebra linear básica (vetores/matrizes), estatística descritiva (média, mediana, desvio padrão), noções de probabilidade. Cálculo básico é útil, mas não obrigatório.
- **Conceitos de dados**: entendimento de tabelas/CSV, tipos de dados, operações básicas (filtrar, agrupar, agregações).

## Ferramentas e versões recomendadas**

O que vamos usar no curso, se quiser instalar em sua máquina.

- **Python**: Python 3.8+
- **Ambientes virtuais**: `venv` ou `conda`
- **Notebooks**: JupyterLab ou Google Colab
