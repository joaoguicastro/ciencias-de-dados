# Ementa

## Módulo 1: Fundamentos de Python para Dados

Foco: Sair do zero em Python e entender a lógica de programação voltada para objetos de dados.

    Sintaxe e Tipagem: Variáveis, strings, listas, dicionários e tuplas.

    Fluxo de Controle: Estruturas condicionais (if/else) e laços de repetição (for/while).

    Funções e Modularização: Criação de funções reutilizáveis e importação de bibliotecas.

    Boas Práticas: como escrever código que você entenderá depois de uma pausa.

## Módulo 2: Manipulação de Dados

Foco: Transformar dados brutos em tabelas prontas para análise utilizando Pandas e NumPy.

    I/O de Dados: Leitura e escrita de arquivos CSV, Excel e introdução a JSON.

    DataFrames: Seleção, filtragem, indexação e fatiamento de dados com Pandas.

    Limpeza de Dados: Tratamento de valores ausentes (NaN), remoção de duplicatas e conversão de tipos.

    Estatística Descritiva: Medidas de tendência central (média, mediana, moda) e dispersão (desvio padrão, variância) usando Pandas e NumPy.

## Módulo 3: Visualização de Dados e Storytelling

Foco: Comunicar descobertas através de gráficos claros e esteticamente agradáveis.

    Matplotlib: Anatomia de um gráfico, customização de eixos, títulos e legendas.

    Seaborn: Gráficos estatísticos avançados (Boxplots, Heatmaps, Vioplots e Regressão Linear).

    Design de Informação: Escolha do gráfico ideal para cada tipo de dado e como melhor definir suas cores.

    Boas Práticas: Como comunicar de maneira clara e evitar gráficos enganosos.

## Módulo 4: Análise Exploratória de Dados (EDA)

Foco: Onde a ciência de dados acontece. Investigar padrões e anomalias.

    Formulação de Hipóteses: Abstrair problemas de negócio em perguntas norteadoras para realizar análise.

    Exploração de Dados: Análise univariada, bivariada e multivariada para entender relações entre variáveis.

    Feature Engineering: Criação de novas variáveis a partir de dados existentes.

    Insights: Sintetizando descobertas para tomada de decisão.

🎓 Projeto Final: Aplicação Prática

Foco: Consolidação do aprendizado em um portfólio real.

    Dataset Real: Escolha de uma base de dados pública (Kaggle, Governo Aberto ou APIs).

    Ciclo Completo: Desde a carga dos dados até a apresentação visual dos resultados.

    Documentação: Escrita de um relatório técnico e organização do repositório seguindo boas práticas.
